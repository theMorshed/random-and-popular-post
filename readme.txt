This is a Lightweight WordPress plugin to Display Random posts, popular posts etc.

== Description ==

Random AND Popular Post plugin allows you to show Your Random posts and Popular Posts in your widgetized area.

You can display posts in widgetized area as much as you want. Also this is very lightweight plugin. It can't complict with another plugin because this plugin is developd using OOP which is most secure way to develop a good plugin. So, You can install this plugin without any hasitation and enjoy with us.
